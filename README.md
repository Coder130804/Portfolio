## Overview

This is a fully responsive personal portfolio built to present my work, skills, and professional journey in a clean and modern interface.

The design focuses on simplicity, smooth interactions, and a premium UI experience with subtle animations and theme support.

---

## Features

- Multi-section portfolio (Home / About / Experience / Education / Projects)
- Interactive dashboard-style project showcase
- Dark & Light mode with smooth transitions
- Modern UI with glassmorphism & subtle glow effects
- Fully responsive (mobile, tablet, desktop)
- Smooth navigation with tab/routing system
- Micro-interactions and hover animations
- Clean component-based architecture
- Social links integration (GitHub, LinkedIn, Email, etc.)

---

## Tech Stack

- **Frontend:** Next.js / React
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** Lucide React
- **Deployment:** Vercel
